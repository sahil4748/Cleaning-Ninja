# Cleaning Ninja - Master Agent Configuration

## Project Identity
You are building CleaningNinja.co - Australia's premium cleaning service website. 
The brand positioning is: "Premium, trustworthy, eco-friendly cleaning services for Australian homes."

## Design Direction
- Style: Clean, minimal, premium (inspired by modern high-quality cleaning service templates)
- Strong whitespace and visual hierarchy
- Color palette: Olive green + beige/warm cream
- Conversion-focused layout with clear CTAs, trust elements, structured flow
- Animations: Smooth, subtle, professional (not flashy or gimmicky)

## Non-Negotiable Rules
1. ALWAYS use the olive + beige color palette defined in design-system.md
2. NEVER use dark gray/black backgrounds - only olive, beige, cream, or white
3. ALWAYS maintain WCAG AA contrast ratios
4. NEVER use generic stock photos without warm overlay treatment
5. ALWAYS implement animations specified in animation-specs.md
6. NEVER skip mobile responsiveness - design mobile-first
7. ALWAYS include the ninja brand elements (logo, subtle motifs)
8. NEVER use more than 2 font families per page

## Output Expectations
- Production-ready HTML, CSS, JavaScript
- Tailwind CSS for styling (consistent with design tokens)
- Framer Motion or GSAP for animations
- Responsive breakpoints: mobile (375px), tablet (768px), desktop (1280px)
- Performance: Lighthouse score 90+ for all metrics
